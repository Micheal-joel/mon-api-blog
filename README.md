 API Blog - TP INF222

Ce projet est une API Backend développée en Node.js avec le framework Express. 
Elle permet de gérer les articles d'un blog via une base de données SQLite.

## Installation
1. Cloner le dépôt : `git clone https://github.com/Micheal-joel/mon-api-blog.git`
2. Installer les dépendances : `npm install`
3. Lancer le serveur : `node app.js`

 Endpoints
- `GET /api/articles` : Liste tous les articles.
- `POST /api/articles` : Ajoute un article (champs : titre, auteur, contenu, categorie).

 Technologies
- Node.js / Express
- SQLite3
- Git / GitHub
